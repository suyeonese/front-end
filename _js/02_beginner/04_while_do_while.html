while과 do while은 반복의 횟수를 예상하기 어렵거나, 반복횟수를 특정할 필요가 없을 때 주로 사용

  • while(condition){ code ... } : condition은 true일 때 code 영역을 반복 수행한다
  • do { code ... } while(condition); : 우선 do{code...} 영역을 실행하고 condition이 true일 때 code 영역을 반복 수행한다 (세미콜론 작성 주의)
  • 조건에 부합하는 반복횟수를 정확하게 예측할 필요가 없을 때 비교, 논리 연산자를 통해 제어 할 수 있다
  • 주의점: 조건이 항상 true이면 무한 반복 수행되기 때문에 조건 설정에 false를 예측할 수 있어야 함
